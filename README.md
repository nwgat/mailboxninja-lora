**Pinout**

**Wemos D1 Mini**
* 3.3V > 3.3V
* GND > GND
* RST > RST
* NSS > D8
* MOSI > D7
* MOSO > D6
* SCK > D5

**LILYGO T-OI 16340 Rechargeable**
* 3.3V > 3.3V
* GND > GND
* RST > RST
* NSS > D8
* MOSI > D7
* MOSO > D6
* SCK > D5
